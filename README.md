Para este proyecto, debes de realizar lo siguiente como fase inicial:
1. Debes de leer trabajo_poo.ufx para que tengas en cuenta todas las clases a crear en el paquete Modelo y Vista, todas las clases las vas a crear con los atributos en formato JAVA, todos en privado, y si hay métodos, estos se realizarán luego cuando se creen los constructores, getters y setters mediante un programa externo; la clase Vista (src/Vista) tiene sus métodos públicos y son estáticos, sólo inicia creando todos los métodos de la clase vista, osea, con nombres.
2. Luego veras que habrán librerías las cuales serán nuestro punto de partida para que las tengas en cuenta.
3. Cuando ya tengamos todas las clases con sus atributos, debemos tener en cuenta la cardinalidad, los arreglos e importar las clases las cuales están relacionadas entre sí.
4. Todas las clases deben de estar serializadas.
5. Se deben utilizar las etiquetas de extends; ArrayList<nombreClase> (para la clase Usa).
6. Se deben crear los atributos de todas las clases Modelo.
7. Para el frontend utilizaremos el GUI de JAVA, pero esto se implementará después.
